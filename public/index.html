Straight talk

Optional on purpose

Pro does not depend on this tool. Create lists under x.com/i/lists, add people yourself, open pro.x.com, make a deck, and pin list columns — all without Guideposts.

This host only exists to help people get started: a first deck plan from signals already on your account, then a short path into Pro. Use it once, use it never, or self-host the MIT code — your call.

How it works

  1. Connect X — short session. Read + list write only. Nothing is posted as you.
  2. Tune dials — how much bookmarks, likes, follows, and replies should count.
  3. Approve lists — review starter private Lists, then create them on your account.
  4. Open Pro — new deck on pro.x.com, Add column → Lists for each XP · list. That is the real product.

Honest costs

The software is free (MIT). X API credits are not. This public host covers 10 full builds per UTC day after a random unlock (1 free build per person per day). Self-host with your own X developer app for more. Optional naming polish uses an xAI API key — X Premium chat Grok is not an API.

Short connections (on purpose)

This host is not a permanent login. Sessions last about 60 minutes, long-lived refresh tokens are not stored, and after lists are created the session ends and X tokens are wiped from the host. Lists already on your X account stay yours — finish the Pro deck without staying connected here.
