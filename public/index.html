User flow

Four steps

  1. Connect Sign in with X for a short session (~60 min). We never post as you. Tokens are wiped after lists are created (or when you disconnect).
  2. Dials Choose how much each signal counts. Zero a dial to ignore it. Presets: Balanced, Saves, Conversations, Follows, DMs.
  3. Lists Review the proposed private Lists (who’s in each and why). Edit names if you want, then create on your account.
  4. Pro Open pro.x.com → new deck → Add column → Lists → pick each list. (Third-party apps can’t create Pro decks via API.)

Path B · ongoing

Your X developer app

Same site — X bills your developer project. No self-hosting required.

  • Create an app on developer.x.com
  • Callback must be exactly:
    https://xpro.howtomovetheneedle.com/api/auth/x/callback
  • Paste OAuth 2.0 Client ID + Secret in the app (not Consumer Key / Bearer)
  • Credentials encrypted here; you can remove them anytime

What we never do
